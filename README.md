# Tic-tac-toe-game

1. Игра крестики-нолики реализована в веб-странице.

Интегрирован Telegram-бот для отправки уведолмлений о результате игры в чат с пользователем. В случае выигрыша пользователь получаетпромокод на скидку и отправляется сообщение в Telegram-бот «Победа! Промокод выдан: [код]»


2. Использованные технологии:

- Frontend:
Структура страницы - HTML5
Стили, дизайн страницы - CSS3
Игра и ховеры - avaScript

- Асинхронное программирование (async/await, Promises)

- Внешние библиотеки и ресурсы:
Font Awesome 6 - иконки для UI элементов
Google Fonts - шрифты Playfair Display и Quicksand
Telegram Bot API - для отправки уведомлений

- Архитектурные решения:
MVC-подобная структура (Model-View-Controller):
Model: игровая логика, состояние доски, проверка побед
View: HTML/CSS представление, модальные окна
Controller: обработчики событий, управление потоком игры

- Разделение на модули:
script.js - основная игровая логика
telegram-bot.js - интеграция с внешним API
style.css - все стили
index.html - структура приложения

- Алгоритмы и структуры данных:
ИИ компьютера реализован с помощью:
Поиск выигрышных ходов (алгоритм проверки паттернов)
Приоритетная очередь ходов (победа > блокировка > центр > случайный)
Минимакс-подобная логика без рекурсии

- Генерация промокодов:
Алгоритм случайной генерации 5-значных кодов
Исключение похожих символов (0/O, 1/I/l)
Использование криптографически безопасного Math.random()

- Интеграции:
Telegram Bot API через Fetch API
RESTful взаимодействие с внешним сервисом
Обработка ошибок и graceful degradation


3. Установка и запуск
- Запуск Веб-страницы:
    Открыть папку проекта в любом из редакторов кода;
    Установить утилиту Live Server или аналогичную (если таковой еще нет);
    ПКМ по файлу index.html - Open with Live Server;
    В открывшемся окне появится визуальная составляющая игры.

- Запуск и настройка Telegram бота:
    В поисковой строке Telegram найти @tic_tac_toe_game_test_bot;
    Запустить бота командой /start ;
    В файле telegram-bot.js заменить значение CHAT_ID на ваш chat_id. Узнать можно с помощью Telegram бота @getidsbot.


4. Программа готова к использованию. Можете играть!
